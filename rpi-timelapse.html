Time-lapse photography on the Raspberry Pi

The Raspberry Pi camera is pretty easy to use, and the still photo quality is high enough to do better than 1080p, so it is a great platform for time-lapse photography. If all you want to do is set it up in one place and take photos, this tutorial is a pretty good place to start.

I had a slightly different idea of what I wanted to use the camera for. I do a lot of electronics projects, and I don't really want to log in and kick off a time-lapse session every time I start a project. I'd rather be able to simply plug the Pi in, hit a button, and have it start recording. I'd especially like to be able to do this without any network access, in case I'm recording at an event. So, with that in mind, I came up with the following list of requirements:

  • Only one wired connection, for power.
  • A stable platform so I can set it up on a table and get a good shot
  • An indicator to tell me when it is actually taking photos
  • A way to dump the photos back to a computer with more storage after the fact
